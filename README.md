Tradease is a full-stack cryptocurrency trading platform designed to deliver real-time market insights and a seamless trading experience. The application integrates live crypto market data to enable users to explore, analyze, and track digital assets efficiently.

The backend is built with Java Spring Boot, providing secure, scalable RESTful APIs, while the frontend is developed using React with Tailwind CSS for a modern, responsive UI. State management is handled using Redux, and Axios is used for efficient API communication. Real-time cryptocurrency data is sourced from the CoinGecko API, ensuring accurate and up-to-date market information.

This project demonstrates end-to-end full-stack development, clean architecture, and modern web development best practices.

Tech Stack:

Backend: Java, Spring Boot

Frontend: React, Tailwind CSS

State Management: Redux

API Communication: Axios

Market Data: CoinGecko API
